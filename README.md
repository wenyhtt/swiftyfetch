# SwiftyFetch

A lightweight, fast system information fetcher written in **Swift**. Similar to `neofetch` or `fastfetch`.

## Build from Source

Ensure you have Swift 6.2 or later installed.

```bash
git clone https://github.com/yourusername/swiftfetch.git
cd swiftfetch
swift build -c release
./build/release/swiftyfetch
```

## Usage

Simply run the executable:

```bash
./swiftfetch
```

The tool will display your system information and an ASCII art logo.

## Requirements

- Swift 6.2 or later
- macOS, Windows, Linux, or other Unix-like systems
